Installation Guides

Installing WampServer

  1. Before execute “wampserver3.2.3_x64.exe”, make sure you have installed all the vc_redist (including x86 and x64) from “all_vc_redish_x86_x64.zip” before executing "wampserver3.2.3_x64.exe".
  2. Execute “check_vcredist.exe” to make sure if all the required vc_redist are installed.
  3. After install vc_redist and WampServer, install “wampserver3_x86_x64_update3.2.4.exe”.

Configuring network for WampServer

  1. Start WampServer

Configuring Server

  1. Install "pdfelement-pro_setup_full5239.exe"
  2. Share the virtual printer with the name of "WondersharePDFelement" in the "Devices and Printers" with the sharing name "Wondershare PDFelement.
  3. Extract and copy the subfolders from “smartorder.zip” into folder “www” where the WampServer is located.
  4. You will get a folder after extracting the zip file, add the folder onto your installed WampServer folder “\bin\apache\apache2.5.6\bin\httpd.exe” onto the “Windows Firewall”, and allowing either “Public” or “Private”.
  5. Allowing “File and printer sharing’ from “Network Sharing Center”, either “Public” or “Private”.

Configuring Database

  1. Login to “MyPhpAdmin”.
  2. Change user password to “hworderstmart”.
  3. Use the sql backup “smartorder.sql” to restore the backup.
